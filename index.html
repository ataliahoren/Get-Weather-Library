Get Weather Library

Library which returns the current weather in a requested city and country.

View on GitHub Download .zip Download .tar.gz

What is the purpose of GET-WEATHER library ?

  • The library allows other developers have an easy and orderly weather data around the world.
  • You can easily get the current weather details in a required location by given name of a city in a country.

More about the GET-WEATHER library..

  • The library uses the RESTful web services from this website : http://api.openweathermap.org
  • The library was developed in the language C#.
  • The invironment is Visual studio 2015.
  • Unit tests is included.
  • The library was developed in the final project of the course 'Programming languages'.

Design pattern implementation in the library :

  • Singletone
  • Factory

What is included in the GET-WEATHER library ?

  • A folder with all classes,interfaces and file needed for the library purpose.
  • Solution file of the library to run on Visual Studio environment.
  • A small program in C# that shows a simple use in the library, how to rerieve the current weather in London, UK.
